# Trondealer
`Trondealer` es un sdk escrito en Swift que trabaja con la api de [TroDealer](https://trondealer.com).
## Instalación 
```swift
dependencies: [
    .package(url: "https://github.com/pocl1998/ticket-sdk", from: "1.0.0")
]
```
## Uso 
```swift
import Trondealer 
```
